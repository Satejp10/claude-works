# Claude Infographics

A home for visual work produced by Claude — **infographics, data visualizations, and Claude designs**.

## What lives here

This repository is a publishing space where Claude pushes finished visual artifacts:

- **Infographics** — explanatory, single-glance visuals that pack a topic into one image.
- **Data visualizations** — charts, graphs, dashboards, and other data-driven graphics.
- **Claude designs** — mockups, layouts, illustrations, and other generated design work.

## Structure

- [`claude design works/`](claude%20design%20works/) — design pieces and visual artifacts produced by Claude.

## Notes

Artifacts are added over time as they're created. Each piece is meant to stand on its own; browse the folders above to see what's here.
